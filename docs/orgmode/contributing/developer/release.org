* Versioning
~cpmdc~ uses semantic versioning on the public C ABI and Cap'n Proto schema
fields.
Additive Cap'n Proto fields and new ABI entry points are minor; breaking field
renumbers or ABI signature changes are major.

* Checklist
1. ~meson test -C build --print-errorlogs~ on the stub/parser configuration.
2. Update ~CHANGELOG.md~ for user-visible changes.
3. Tag ~vX.Y.Z~ when publishing.

0.1 focuses on Cap'n Proto structure, C ABI, and the ISO_C embed shell.
OpenCPMD energy/force evaluation is a later embed milestone.
